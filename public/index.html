Dokümantasyon

API uç noktalarını kullanarak kolayca verilerinize ulaşabilirsiniz:

Örnek Uç Nokta

GET https://api.synex.com/v1/data?param=value

Parametreler

  • param: Talep ettiğiniz verinin detayını belirler.
  • value: Talep edilen değerin detayını içerir.

Örnek Yanıt

{
  "status": 200,
  "message": "Başarılı",
  "data": {
    "key": "örnek değer"
  }
}
