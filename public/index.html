Nasıl Kullanılır?

Bir bitiş noktası kullanmak için istenen bitiş noktasına aşağıdaki parametrelerle bir GET isteği gönderin:

    
message: Talebinizi girin

Cevap

Verilen yanıt JSON formatındadır:

{
  "status": 200,
  "creator": "Synex",
  "data": {
    "response": "Talebinizin Yanıt Sonuçları"
  }
}
